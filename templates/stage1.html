Lesson 2 - Basics Of HTML

HTML - HYPER TEXT MARK UP LANGUAGE

HTML is the laguage of web. It is used to describe the web page or specifically the layout and content of a web page.

HTML is made up of following:-
  1. Text Content
  2. Tags- Mark up is done using tags
  3. Images & Videso - Has references to videos,images, animations etc.
  4. Links - links to other pages

HTML Elements

HTML markup is done using tags. Tags are like instructions telling the browser how to present a content. It has an opening tag and a closing tag encasing the content to be dispalyed as per the instructions provided in the tag.

The syntax is as shown: <tag name> ...content... </tag name>

This combination is also referred to as Element.

The HTML Document structure are further represented ina tree like manner. The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them.

You can draw this relationship like a family tree. My mother had multiple children.So did her mother, and so on...In a browser, this structure shows up as a series of nested boxes.There are boxes inside of boxes inside of boxes, and so onPofessional programmers also refer this as "Document Object Model".


HTML and boxes

When you read an HTML document as text, you see a wave of changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested it's corresponding "box" is.

Lesson 4 - Adding CSS Style to HTML Structure

CSS

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements (either inline, Internal or External). This is done by giving similar HTML elements the same class and then specifying the style that should apply to that class.CSS saves a lot of work by controlling multiple page layout at once. It thus helps in avoiding repetition of code & you can change the oulook of your whole page by just changing or adding CSS Sheet.
A CSS rule set consists of selector &declaration. In the declaration block we can define the property & value of each property. A declaration can contain many properties at a time as shown below:


CSS Styling can be done in 3 differnt ways:
  1. INLINE STYLE- This uses style attribute in the same line as HTML code as shown below:
    <h1 style="color:blue;margin-left:30px;"> .This is a heading. </h1>
  2. INTERNAL STYLE- If the style is unique to the page, we can use Internal style. This is defined within the sytle element inside the head section of the HTML page as below: 
                              <style>
                          
                                 body {
                              background-color: linen;
                                  }
                             h1 {
                              color: maroon;
                              margin-left: 40px;
                                } 
                          
                              </style>

  3. EXTERNAL STYLE- Using external sheet we can re-define the whole web page by just changing one file. The first step is to include the external fine using the link element inside the head of the HTML page: 
                              <head>
                          <link rel="stylesheet" type="text/css" href="mystyle.css">
                          </head>

Avoid Repetition

Avoiding repetition is important for a variety of reason

Avoiding Errors: When programmers don't avoid repetition, they will often have to do the same thing over and over. This may mean copy and pasting certain style attributes into many HTML elements or rewriting the same code (sometimes with minor differences) many times. This can lead to errors when the programmer decides to make a change to something. If they don't dilligently make that same change everywhere the repeated code appears, problems will arise.

Consistency: Maintain same style by defining a class, then make changes only to the class described around the content.By avoiding repetion, it's now very easy for me to make changes to this style.


The Box Model

In CSS, the term "box-model" is used while designing the layout of the page. It essentially wraps around all HTML elements and consists of : Margins, Border, Padding and Content. The image below ilustrates the box model:


Content - The content of the box, where text and images appear
Padding - Clears an area around the content. The padding is transparent
Border A border that goes around the padding and content
Margin - Clears an area outside the border. The margin is transparent
